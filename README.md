# 午餐抽卡機 🎴

一個精美的日系極簡風格單頁 Web 應用，幫你決定今天午餐吃什麼！

## 功能特色

- 🎨 日系極簡風格設計
- 🌈 柔和的漸層背景
- 🎴 精美的卡牌翻轉動畫
- 🎉 Confetti 粒子特效
- 🍜 10 種預設午餐選項

## 如何啟動

1. 安裝依賴（如果還沒安裝）：
   ```bash
   npm install
   ```

2. 啟動開發伺服器：
   ```bash
   npm run dev
   ```

3. 在瀏覽器中打開顯示的本地地址（通常是 `http://localhost:5173`）

## 如何構建

構建生產版本：
```bash
npm run build
```

構建後的檔案會在 `dist` 資料夾中。

## 技術棧

- Vite
- React
- Tailwind CSS
- canvas-confetti
